SETTING UP SSH

This is done toensure a secure connection between local machine and Github

GENERATE SSH KEY

Open Git bash terminal and type the following command ssh-keygen -t ed25519 -c "your-email@example.com" you will be prompted to to enter file to save the key just press ENTER to save it in default location then you will be prompted to enter a passphrase you can either enter one or just press ENTER twice to skip

ADD SSH KEY TO SSH-AGENT

Start the ssh-agent in the background by running the following command eval "$(ssh-agent -s)" then add your SSH private key to the ssh-agent with the command ssh-add ~/.ssh/id_ed25519

ADD SSH KEY TO GITHUB ACCOUNT

Copy the SSH key to your clipboard with the command clip < ~/.ssh/id_ed25519.pub then go to your Github account settings navigate to SSH and GPG keys click on New SSH key paste your key and save

CREATE A REPOSITORY

Click the "+" icon (top right) New repository

Name your repository (e.g., "yourname-unipod-name

Choose "Public" or "Private"

Click "Create repository"

CLONE THE REPOSITORY LOCALLY

In your repository on GitHub, click the "Code" button and copy the SSH URL

Open Git Bash on your local machine and navigate to the directory where you want to clone the repository

Run the command git clone then paste SSH key copied from github

Bringing the Cloud to Your Computer

Cloning creates a local folder connected to your GitHub repo
