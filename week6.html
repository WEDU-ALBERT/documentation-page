Introduction

Laser cutting is a digital fabrication technique that uses a high-powered CO₂ laser beam to cut or engrave material such as acrylic, MDF, plywood, cardboard, and leather. The machine follows vector paths generated from software like SolidWorks, allowing for accurate and repeatable cuts.

Process Documentation

1. Setting Up the CO₂ Laser Machine

  1. Place your material on the cutting bed.
  2. Switch on the water chiller (must reach correct (room temperature)).
  3. Check water flow to the laser tube.
  4. Turn on the extractor fan to remove fumes.
  5. Switch on the main laser power.
  6. Home the machine using the control panel.
  7. Adjust the focus height using the focusing tool.

2. Preparing the Drawing in SolidWorks

  1. Design for Assembly (DFMA)
  2. Draw a series of designs which assemble to make one product in my case an outdoor bench
  3. Fully define sketches and ensure accuracy by assembling parts in assembly.
  4. Save your parts: Click > File > Save As > DXF repeat for all parts of the assembly.
  5. NB: Choose Face when saving parts

3. Preparing DXF File for Cutting

The Full Spectrum CO2 laser-cutting machine operates through a network-based communication system. Its control software is embedded within the machine's internal architecture, eliminating the need for external installation. Access to the machine's interface is achieved by locating and entering its designated IP address, which launches the integrated web-based control environment. Connection between the computer and the laser-cutting system is established via an Ethernet cable. Provided that both devices are connected to the same network, stable communication is maintained, enabling seamless data transfer and operational control of the machine.

Steps to access the laser cutting software:

  1. On the control panel turn on the machine using a key
  2. Check network settings to find the IP address of the laser machine
  3. Open browser to access laser cutting software compatible with our CO₂ laser
  4. Search for laser machine's IP address
  5. Software interface
  6. Import the DXF file

    7. Drag and drop DXF or PDF file into software workspace

    Check if the scale matches your original dimensions.

    Assign colors:

    • Red → Cutting
    • Black → Engraving

    Set parameters:

    • Cutting: Power 60–80%, Speed 8–12 mm/s
    • Engraving: Power 20–30%, Speed 200–300 mm/s

4. Cutting the Job

  1. Run parameter check to ensure work piece is within the material
  2. Monitor the cutting process constantly.
  3. Wait for fumes to clear after cutting.
  4. Remove the finished parts.

Reflection

Learning to operate the CO₂ laser cutter was an exciting and highly practical experience. I learned how different materials respond to changes in speed and power, and I gained confidence in generating DXF files directly from SolidWorks. Understanding safety protocols and machine setup helped me appreciate how sensitive the machine is to cooling, focusing, and airflow. This skill will be extremely useful in prototyping, product design, and fabrication work.
