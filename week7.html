Quick Summary

    1. 3D Printing Methods

      • FDM (Fused Deposition Modeling)
        Uses melted plastic filament (PLA, ABS, PETG, etc.)
        Prints by depositing material layer by layer
        Good for prototypes, functional parts,low-cost printing and Reduces redundancy of parts
      • SLA (Stereolithography)
        Uses UV light/laser to cure liquid resin
        Produces very smooth and highly detailed prints
        Good for miniatures, dental models, jewelry, and precision parts

    3D Printing – Test Print Rules

    Rules for Doing a Test Print on a 3D Printer

    Prepare the Machine

    • Switch on the printer and check that all components power on correctly.
    • Confirm filament type (PLA, PETG, ABS, etc.).
    • Load filament and purge until clean extrusion appears from the nozzle.

      • Level the Bed

      • Perform automatic bed leveling or calibaration before every test print.

      Prepare the Print Bed

      • Clean bed surface of dust and debris.
      • Use mild soap + water for smooth PEI sheets.
      • Apply glue stick if needed for better print adhersion.
    • Temperature tower (220deg for PLA filament which i used for testing)

    Slice with Correct Settings

    • Nozzle temperature: follow filament label.
    • Bed temperature: follow filament label.
    • Layer height: 0.2 mm recommended for test prints.
    • Print speed: 50–60 mm/s for accurate testing.
    • Cooling: ON for PLA, reduced for ABS/PETG.

    Load and Check G-code

    • Slice the model and export as .Gcode.
    • Ensure the G-code matches the printer model.
    • Load file to SD card or USB stick.

    Monitor the First Layer

    • Never leave the printer during the first layer.
    • Check adhesion and smooth extrusion.
    • Stop immediately if gaps or lifting appear.

    Safety Rules

    • Do not touch the hotend or heated bed while hot.
    • Keep away from flammable materials.
    • Ensure ventilation when printing ABS/ASA.
    • Do not leave the machine unattended for long periods.

    Evaluate the Print

    • Check layer consistency.
    • Measure accuracy using calipers.
    • Look for under-extrusion, warping, or stringing.
