Introduction

This week focused on introducing 3D design and assembly using SolidWorks. The main objectives were to learn how to create 3D models, assemble multiple components, and generate technical drawings. SolidWorks is a powerful CAD software widely used in engineering and design fields for creating detailed 3D models and assemblies.
